# The Curator

**The Curator** is a premium, high-end editorial platform built for the modern digital mastery era. It moves beyond the "template" look of modern tech blogs, offering a sophisticated experience that marries Swiss minimalist structural integrity with the tactile warmth of a premium print magazine.

## 🏛️ The Editorial Precision Framework

This project is built on a custom design system that prioritizes hierarchy, whitespace, and tonal depth over rigid borders and generic componentry.

- **Intentional Asymmetry**: Hero sections and article grids utilize "breathing room" as a functional element.
- **Tonal Layering**: Zero 1px borders. Boundaries are defined solely through background color shifts.
- **Dual-Font Strategy**: A sophisticated dialogue between **Manrope** (Interface/Tech) and **Newsreader** (Narrative/Editorial).
- **Glassmorphism**: Floating elements utilize high-blur backdrops to simulate fine-paper layering.

## 🚀 Technology Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Typography**: Goole Fonts (Manrope & Newsreader)
- **Imagery**: Custom high-resolution AI-generated assets
- **Icons**: Custom SVG system

## 📂 Features

- **Dynamic Category Routing**: Specialized results pages for AI, Tech, Programming, Design, and Freelancing.
- **Contextual Sidebars**: Trending insights and contributor highlights tailored to each domain.
- **Advanced Search**: Case-insensitive filtering across titles, excerpts, and taxonomy.
- **Asymmetrical Feed**: A curated article grid that balances density with readability.

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
